# Hanam Online Prayer Room

하남교구 성도들을 위한 모바일 중심 온라인 기도실입니다.  
기도 제목 작성, 공개 기도실 열람, 함께 기도하기(카운트) 기능을 제공합니다.

## 주요 기능

- 홈 화면에서 `중보 기도실 입장` / `기도 제목 작성` 진입
- 기도 제목 작성:
  - 이름, 전화번호, 기도 내용, 공개 범위(전체 공개/교역자만 보기)
  - 전화번호 자동 포맷팅 (`010-1234-5678`)
  - 동일 조건 중복 시 기존 유지/새 내용 교체 선택
- 중보 기도실:
  - 공개(`is_public = true`) 기도 제목 최신순 열람
  - `기도하기` 토글로 `prayed_count` 증감
  - 로컬스토리지 기반 개인 기도 여부 기억
- 시니어 사용성 중심 UI:
  - 큰 타이포 유지
  - 터치 타깃/포커스/모션 접근성 개선

## Supabase 설정

현재 프로젝트는 `supabaseClient.ts`에 Supabase URL/Anon Key를 직접 사용합니다.  
실사용 시 아래 값을 본인 프로젝트 값으로 교체하세요.

- `/supabaseClient.ts`
  - `SUPABASE_URL`
  - `SUPABASE_ANON_KEY`

### 필요 테이블 스키마

```sql
create table public.prayers (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  name text not null,
  phone text not null,
  content text not null,
  is_public boolean default true,
  prayed_count integer default 0
);
```

이미 테이블이 있다면:

```sql
alter table public.prayers add column prayed_count integer default 0;
```
